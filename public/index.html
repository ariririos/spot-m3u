Spotify To M3U

This website converts a Spotify playlist into the M3U(8) format.

Because of Spotify's API limitations only the first ~50 playlists will show and only the first ~100 tracks will be included in the playlist file. These issues will be fixed eventually. If you want to help check out the repository.

1. Login with Spotify

2. Specify the formatting of your MP3's

The formatting cannot be changed at the moment

3. Choose a playlist to download

You need to login first

4. Wait for the conversion to complete

5. Download your playlist
